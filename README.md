A collection of sorting algorithms written in Java for [Growing with the Web][1].



## Dependencies

Java (JRE, JDK) and [Gradle](https://gradle.org/) are required to build and test this project.



## Build

```bash
gradle build
```



## Test

```bash
gradle test
```
